支持：Java、Python、JavaScript、Go、Perl、PHP、Ruby、Swift、TypeScript、SQL、Shell ...  

        在左下角的输入框中输入你想实现的前端需求，然后点击提交，剩下的交给马克！ 

        马克写的代码会放到右上空白区域、代码运行后的效果会在左上区域展示。 

        如：绘制一个中国九九乘法表 、  绘制一首李白的诗
      

      
如果用户在五分钟之内无任何操作，将自动清空当前用户数据

      
如果马克给出的代码在渲染期间报错了，需要刷新页面后方可继续使用。
